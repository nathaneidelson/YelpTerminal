This is a simple ruby script which can be ran in terminal. It returns the top 10 results as they
would appear on Yelp. For example, "yelp thai in Palo Alto" lists the top 10 Thai resturants near Palo Alto.

Requirements:
HTML parser "Nokogiri" installed as a ruby gem (http://nokogiri.org/tutorials/installing_nokogiri.html)
